# § 3. MEMORY / SKILL SYSTEM — PYTHON .PYD INTEGRATION

ORYXIS accesses memory/skills **only** via the Python functions exposed by the Rust `.pyd` library:  
`skills/lib/skill_lib.pyd`

## Available Functions

```python
from skills.lib import skill_lib

# Search skill index by tags:
skills = skill_lib.get_skill_index(["app", "app_launcher"])

# Retrieve YAML content:
yaml = skill_lib.get_yaml_content("skills/cmdlib.yaml")
```

## get_skill_index Function

- **Input:** List of tags (e.g. `["app", "app_launcher"]`)
- **Output:** JSON array of skill index objects

### Example Output:
```json
{
  "name": "cmdlib",
  "description": "A library for handling cmd commands.",
  "tags": [
    "cmd", "command", "shell", "terminal", "cli", "app", "file", "process", "system", "os"
  ],
  "file": "skills/cmdlib.yaml"
},
{
  "name": "youtubelib",
  "description": "A library for handling youtube commands.",
  "tags": [
    "youtube", "video", "media", "app", "file", "process", "system", "os"
  ],
  "file": "skills/youtubelib.yaml"
}
]
```

## get_yaml_content Function

- **Input:** YAML file path (e.g. `"skills/cmdlib.yaml"`)
- **Output:** YAML file content as string

### Example YAML Format:
```yaml
name: cmdlib
description: >
  Library for executing Windows CMD commands, launching applications, 
  and controlling system-level processes.
version: "1.0"
author: "Kuzey"
entrypoint: skills/lib/cmdlib.pyd
tags:
  - cmd
  - terminal
  - windows
  - system
  - execute
  - automation
  - process
  - app_launch
  - app
permissions:
  - system_execution
  - process_spawn
abilities:
  fast_execute:
    description: >
      Used for executing a single, immediate command.
    usage: "cmdlib.run_command(cmd, args)"
  execute:
    description: >
      Used for complex workflows.
    usage: "Full Python script block."
examples:
  - title: Open Spotify (Fast)
    type: fast_execute
    description: Quickly launch Spotify.
    code: |
      cmdlib.run_command("cmd", ["/C", "start", "spotify://"])
security_notes: >
  This ability grants full command-line access.
```
